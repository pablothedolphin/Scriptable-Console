Scriptable Console Documentation

The Scriptable Console is an open source utility based on Scriptable Framework for adding a totally scriptable developer console to the Unity runtime much like what is found in Minecraft or Skyrim. (Source code can be found here)

  • For detailed steps on how to make use of all the tools and workflows provided, navigate to the Manual tab.
  • For a detailed breakdown of all namespaces, types, methods and properties, navigate to the Scripting API tab.

The core design behind this utility was inspired by project by from DapperDino on GitHub.
